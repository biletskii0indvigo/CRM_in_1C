## ℹ️ Правила работы с репозиторием

1. **Источник истины:** `MoM/Summary_Meet_0_to_9.md` > `docs/*` > `params/*`.
2. **Документы с гипотезами:** `docs/STATUS_DISCLAIMER.md` — это v0.1.0/v0.2.0, не утверждённое ТЗ.
3. **Архивы** (не редактировать): `demo app/version 2/docs_archive_v0.1.0/`, `roadmap/archive/`.
4. **Новые требования** из встреч → сначала в `MoM/`, затем в `docs/` через обновление версии.
5. **Имена объектов 1С** — только из `docs/DATA_MODEL.md` (после Discovery — из v0.3.0).
6. **Бизнес-идеи вне MVP** → `backlog/`, не в `docs/`.
