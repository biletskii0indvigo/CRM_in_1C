## 5. Ключевые риски и допущения

### Риски
1. **Инфляционный риск** — операционные затраты растут на 7.5% ежегодно
2. **Комплаенс-риск** — возможные изменения в 152-ФЗ и требованиях ЕГИСЗ
3. **Риск масштабирования** — рост пользовательской базы на 5%/год может увеличить затраты на поддержку
4. **Кадровый риск** — стоимость замены врача 750,000 руб. (важно для Clinical Champion роли)

### Допущения
- Resource costs стабильны в течение 5 лет (без индексации зарплат)
- Cloud-архитектура исключает капитальные затраты на железо
- Не учтены потенциальные savings от автоматизации (soft benefits с коэффициентом 0.85)

## 8. Приложения

### A. Формулы расчёта

**NPV Calculation:**
```
NPV = Σ (Cash Flow_t / (1 + r)^t)
где r = 12% (discount rate), t = year
```

**Inflation Adjustment:**
```
Cost_Year_n = Cost_Year_1 × (1 + inflation_rate)^(n-1)
где inflation_rate = 7.5%
```

**FTE Cost:**
```
Role Cost = FTE_share × FTE_Fully_Loaded_Cost
где FTE_Fully_Loaded_Cost = 1,200,000 руб./год
```
