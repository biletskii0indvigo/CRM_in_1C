# Стратегия увеличения среднего чека и LTV в 1С:Медицина «Улучшенная CRM»

## Приложение к программе лояльности и развития продаж

**Версия:** 1.0  
**Дата:** 29.04.2026  
**Для сети клиник «Институт Движения»**

## ВВЕДЕНИЕ

Настоящий документ разработан на основе технического задания на внедрение «Улучшенной CRM» и содержит практические рекомендации по работе с пациентами для увеличения среднего чека и пожизненной ценности клиента (LTV).

Документ включает:
- Конкретные действия с покупателем медицинских услуг
- Поэтапный чек-лист внедрения практик
- Интеграцию с функционалом 1С:Медицина
- Ожидаемые результаты и KPI

## ЧАСТЬ 3. ИНСТРУМЕНТЫ 1С:МЕДИЦИНА «УЛУЧШЕННАЯ CRM»

### 3.1. Обязательные поля после «Финальной консультации»

| Поле | Тип | Варианты | Зачем нужно |
|------|-----|----------|-------------|
| `тип_ведения` | Выпадающий список | контрольный_визит, поддерживающее_лечение, контроль_диагностики, комбинированное | Планирование типа взаимодействия |
| `подтип_ведения` | Выпадающий список | 3мес/6мес/12мес, массаж/лфк/физио, мрт/анализы | Детализация плана |
| `дата_след_контакта` | Дата | ДД.ММ.ГГГГ ≥ сегодня | Триггер для авто-задач |
| `формат_контакта` | Выпадающий список | визит, процедура, диагностика | Планирование ресурсов |
| `комментарий_для_пациента` | Текст (до 280 симв.) | Без мед. терминов | Скрипт для куратора |
| `ответственный_куратор` | Сотрудник | Из справочника | Назначение ответственного |

**Валидация:** Система блокирует закрытие этапа, если хотя бы одно поле не заполнено.

### 3.2. Воронки и статусы

**Основная воронка продаж:**
```
1. Записан → 2. Пришел → 3. Прием проведен → 4. План лечения → 
5. Передан куратору → 6. Презентация плана → 7. Продажа (100%/частичная/пауза/отказ)
```

**Воронка лечения:**
```
1. Старт лечения → 2. В процессе → 3. Промежуточная консультация → 
4. Финальная консультация → 5. Ведение назначено → 6. Завершен курс
```

**Разветвление после презентации плана:**
- **Продажа 100%** → Старт лечения (полный курс)
- **Продажа частичная** → Частичное лечение + апселл позже
- **Пауза** → Сценарий дожима 0-3-7 дней
- **Только диагностика** → Промежуточная консультация
- **Отказ** → Реактивация через 30-60 дней

### 3.3. Автоматические сценарии

| Событие | Авто-действие | Срок | Ответственный |
|---------|---------------|------|---------------|
| Завершение «Финальной консультации» | Создание отложенной сделки | Сразу | CRM |
| Дата_след_контакта - 7 дней | Задача куратору: «Плановый контакт» | За 7 дней | Куратор |
| Пациент не записался после презентации | Запуск сценария дожима День 0 | Сразу | Куратор |
| День 3 сценария дожима | Авто-SMS/WhatsApp с напоминанием | День 3 | CRM |
| День 7 сценария дожима | Повторный звонок куратора | День 7 | Куратор |
| День 14 без результата | Эскалация РОП | День 14 | РОП |
| День 30 без результата | Перевод в сегмент «Долгосрочная реактивация» | День 30 | CRM |

### 3.4. Сегментация пациентов

| Тип пациента | Критерий | Подтипы | Для чего используется |
|--------------|----------|---------|----------------------|
| `первичный` | Нет истории визитов | — | Базовая статистика |
| `повторный_в_лечении` | Активный курс лечения | — | Контроль процесса |
| `реактивированный` | Последний визит ≥180 дней назад | сам_вернулся, обзвоном, реклама, рекомендация | Оценка каналов возврата |

**Дополнительные поля:**
- `география`: местный / иногородний
- `источник_возврата`: звонок / сообщение / реклама / рекомендация
- `психотип_лида`: экономный / тревожный / решительный / исследователь

### 4.2. Качественные улучшения

### А1. Скрипт куратора: День 0 (пациент не записался после презентации)

```
📞 Звонок пациенту

«Здравствуйте, [Имя Отчество]! Это [Имя куратора] из клиники «Институт Движения».

Вижу, что доктор [ФИО врача] рекомендовал вам [кратко: комментарий_для_пациента].
Подскажите, удалось ли ознакомиться с планом лечения?

[Если «Да, но не решил»]
→ Понимаю, решение важное. Что именно вызывает сомнения? 
  Может быть, неудобное время или финансовые вопросы?

[Если «Нет, не смотрел»]
→ Давайте я кратко расскажу, почему это важно именно сейчас... 
  [2-3 предложения из комментария врача]

[Работа с возражениями]
• «Дорого» → Можем разбить на этапы или предложить рассрочку
• «Нет времени» → Есть вечерние часы или субботний приём
• «Хочу подумать» → Конечно! Давайте я перезвоню через 2 дня, хорошо?

[Закрытие]
→ Отлично! Тогда предлагаю [дата и время]. Записать вас?
```
