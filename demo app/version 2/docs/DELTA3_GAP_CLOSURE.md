## 5. Архитектурный паттерн Δ3

Использован паттерн **«wrapper / override»** — без правки исходных функций:

```js
// Сохранить оригинал → переопределить с расширением
const _origFn = someFn;
someFn = function(){ /* extra logic */ _origFn(); /* more */ };

// Или полностью заменить
fn = function(...){ /* new impl */ };

// Или дополнить render-результат:
const _orig = renderFoo;
renderFoo = function(){
  let html = _orig();
  return html.replace(/anchor/, banner + 'anchor');
};
```

Это:
- Не ломает существующие тесты
- Изолирует Δ3-логику в одном файле
- Позволяет легко откатить целиком (просто убрать `<script src="delta3.js">`)
- Облегчает регрессионный прогон
