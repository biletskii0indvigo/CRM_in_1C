# PARAMETERS_REGISTER — Реестр параметров CRM «Институт Движения»

_Параметров: 150 в 11 функциональных блоках._

_Платформа: 1С:Предприятие 8._


## Условные обозначения

- **Приоритет MoSCoW:** Must (обязательно) / Should (важно) / Could (желательно) / Future (на перспективу)
- **Уверенность:** Высокая (явно в документе) / Средняя (выведено из контекста) / Низкая (требует подтверждения)
- **Документ-источник:** идентификатор `DOC-XX` см. в `INDEX.md`
